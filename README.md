#COINBASE - DEMO
##Setup:
- Install python
- Install pip
- Install library: pip install -r requirement.txt

##Set env:
###Ubuntu:
- export APP_KEY = <APP_KEY>
- export API_SECRET = <APP_SECRET>
- export APP_PASS = <APP_PASS>
###Windows:
- setx APP_KEY = <APP_KEY>
- setx API_SECRET = <APP_SECRET>
- setx APP_PASS = <APP_PASS>
